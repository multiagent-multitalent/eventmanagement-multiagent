## WLAN-Anforderungen

| Spezifikation | Anforderung |
|---|---|
| Gleichzeitige Geräte | Mind. 300 (Teilnehmer + Helfer + Backup) |
| Bandbreite Minimum | 500 MBit/s Download gesamt |
| Latenz | <20ms für stabile Verbindungen |
| Netzwerke | 2 separate SSIDs: Teilnehmer + Veranstalter/Technik |
| Passwort-Konzept | Einfaches Passwort täglich wechseln oder Open Network |
| Coverage | 100% aller Räume inkl. Foyer |

**Falls Venue kein ausreichendes WLAN bietet:**
- Mobiler WLAN-Router (LTE/5G) als Backup
- Temporäre Fiber-Leitung anfragen

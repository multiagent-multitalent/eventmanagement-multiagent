# Beispiele / Examples

Dieses Verzeichnis enthält Demo-Outputs, die zeigen, was der Orchestrator für verschiedene Events generiert.

## Verfügbare Demos

| Demo | Event | Beschreibung |
|---|---|---|
| [`aitd-2026/`](aitd-2026/) | AI Transparency Days 2026 | Konferenz mit 125 Teilnehmern, 3 Tage, Nürnberg |

## Zweck

Jeder Demo-Ordner enthält:
- Die verwendete `config/event.yaml`
- Alle generierten Planungsartefakte in `workstreams/`
- Das Planungs-Dashboard und den Entscheidungslog

Diese Beispiele dienen als Referenz und Inspiration. Für dein eigenes Event nutze das **Hauptverzeichnis**:

→ **[QUICKSTART.md](../QUICKSTART.md)** – In 3 Schritten zur fertigen Event-Planung

## Kritischer Pfad

Die folgenden Aufgaben sind **zeitkritisch** und bestimmen den Projekterfolg. Verzögerungen hier gefährden das Gesamtprojekt:

```
[HEUTE 31.03.] → [Kick-off 01.04.] → [Venue-Anfragen 05.04.] → [Venue gebucht Mai] → [Catering beauftragt Juni]
                                   → [CfP live 30.04.] → [CfP-Deadline 15.06.] → [Programm final 31.07.]
                                   → [Registrierung offen 01.05.] → [Early Bird Ende 30.06.]
```

**Höchste Risiken auf dem kritischen Pfad:**
1. 🔴 **Venue (Oktober = Nürnberger Messe-Saison)** → sofort anfragen, nicht warten
2. 🔴 **Keynote-Speaker** → früh anfragen (mind. 6 Monate vorher = bis April)
3. 🔴 **Förderantrag** → Fristen beachten; falls BMBF, Vorlauf >3 Monate einplanen
